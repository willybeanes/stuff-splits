# Stuff Splits

MLB pitch-quality model — Stuff+ / Location+ / Pitching+ grades by pitch type
and batter handedness, plus a FanGraphs comparison view.

Split out from [battery-splits](https://github.com/willybeanes/battery-splits).
Shares the same Supabase project (different tables). See
`pitch-quality-model/README.md` for the model pipeline.
